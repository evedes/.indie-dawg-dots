# indie-dawg-dots

These are my 2025 dots.

Beware, they're a work in progress.

## How to install

1. clone this repository
2. edit config.sh with your information
3. run `./indie-dawg-dots.sh install`

## What does it do?

### Configs

1. generates a .gitconfig file from a template
2. sym links kitty config

### Packages

1. kitty
